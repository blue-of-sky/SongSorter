# SongConverter

SongSorterの使い方
esegitでtjaをコピーして、そのtjaの順番を本家と同じ順番にします。
1. AddSongsタブで曲を追加します。(gitをインストールしていない場合はhttps://git-scm.com/からインストールしてください)
2. SongSorterタブで先ほど追加したSongsフォルダを選択して、その下はコピー先の太鼓シミュフォルダを選択します。
3. 並び替え開始ボタンを押します。
すると曲の順番が本家通りになります

DanGeneratorの使い方
太鼓wikiの段位ページのURLを使い、段位の名前と条件を使ってTaikøNauts等で使える形の段位ファイルにします。
1. 太鼓wikiの段位ページのURLを入力します。(例であげるとhttps://wikiwiki.jp/taiko-fumen/%E6%AE%B5%E4%BD%8D%E9%81%93%E5%A0%B4)
2. 出力するフォルダ名を決めます。(DanListsフォルダの中に生成されるフォルダ名です。)
3. 太鼓シミュのフォルダを選択します。
4. 段位生成ボタンを押します。
すると本家と同じ段位が生成されます
