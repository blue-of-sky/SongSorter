# SongConverter

SongSorterの使い方
esegitでtjaをコピーして、そのtjaの順番を本家と同じ順番にします。
1. AddSongsタブで曲を追加します。(gitをインストールしていない場合は https://git-scm.com/ からインストールしてください)
2. SongSorterタブで先ほど追加したSongsフォルダを選択して、その下はコピー先の太鼓シミュフォルダを選択します。
3. 並び替え開始ボタンを押します。
すると曲の順番が本家通りになります

DanGeneratorの使い方
太鼓wikiの段位ページのURLを使い、段位の名前と条件を使ってTaikøNauts等で使える形の段位ファイルにします。
1. 太鼓wikiの段位ページのURLを入力します。(例であげると https://wikiwiki.jp/taiko-fumen/%E6%AE%B5%E4%BD%8D%E9%81%93%E5%A0%B4 )
2. 出力するフォルダ名を決めます。(DanListsフォルダの中に生成されるフォルダ名です。)
3. 太鼓シミュのフォルダを選択します。
4. 段位生成ボタンを押します。
すると本家と同じ段位が生成されます

DanConvertorの使い方
OpenTaiko等のtja段位をTaikøNauts等で使える形の段位ファイルにします。
1. 変換するtjaが入っているフォルダを選択します。
2. 出力するフォルダ名を決めます。(DanListsフォルダの中に生成されるフォルダ名です。)
3. 太鼓シミュのフォルダを選択します。
4. 変換実行ボタンを押します。
すると段位が生成されます

※難易度の指定はtjaの仕組み上難しかったので、難易度の指定は"danSongs"の"difficulty"を変えて指定をお願いします(0:かんたん,1:ふつう,2:むずかしい,3:おに,4:おに裏)
